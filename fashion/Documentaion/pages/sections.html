Sections

We care about semantic code that's why we don't use a lot of classes when writing HTML markup, instead, we give every section of the page a special class so we can style that section through our sass/css files. Let's try an example here, If you have 'about' page and you have a section called 'features' in that page, that section would have the class .features inside a wrapper with the class .about and this is the general idea of styling all sections on all templates.

Banner Section

Here is the code for the Banner section that used in the homepage, as you see, it has a spacing class. Some sections have a title representing the section and we call it section headline and it usually has the class .section-headline.

Example


<section class="banner">
    <div class="container">
        <div class="row">
            <div class="col-md-10 col-md-offset-1">
                <h2>Campaign Model Hunt 2016</h2>
                <a href="#" class="btn btn-default btn-light">Campaign</a>
                <a href="#" class="btn btn-bordered btn-light">Purchase</a>
            </div>
        </div>
    </div>
</section>
