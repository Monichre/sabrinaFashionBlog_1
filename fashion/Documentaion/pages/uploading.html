Uploading

1. Customize

The first step after downloading the template is to customize it, and for that you need a text editor and we prefer Sublime text or Atom. You can view this Template in any web browser from your desktop computer. Because the files are written in HTML, you do not need an internet connection in order to display or edit the Template. The exception is the contact form which functions based on the PHP coding language and requires a server (although you can setup a local server with wamp) to send messages.

2. Start FTP Client.

Start your FTP client (We persoanlly use Filezila which is free) and login to your server. You will need the following information:

Your server name (often times this will be your domain name)
Username
Password

3. Upload the files

Locate the domain folder you wish to upload your Theme files to on your server. You are going to click and drag every file that you have labeled .html, .css, .php, .js into your FTP client.

4. Hello World.

Once the files are done uploading, rename the file you want to be your home page as 'index.html'. Excellent, now you have an awesome website.
