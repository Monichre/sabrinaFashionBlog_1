Overview

Deliver Fashion is a creative minimal template made for fashion agencies, fashion lovers, deigners and those who need a creative way to showcase their love to fashion.

Responsive Design

Deliver Fashion is fully responsive thanks to the Bootstrap grid and it was tested on real devices to make sure that it work beautifully across all devices.

Built with Sass

Deliver Fashion is a high quality product that's why we use the best preprocessor for the job. Sass files are included if you needed it.

Updates & Updates Detail

Fashion agencies should have a updates page to write articles about their business and work, that's why we have included updates and updates detail pages.

Lookbook Detail

Lookbook Detail and About pages are also included in the template.

Contact Form

Active contact form are included in the template to enable your visitors to contact you and send you messages.

Browser Compatability

Deliver Fashion supports all the major browsers including IE9+, Chrome, Safari, Firefox. It's also tested on real devices to make sure it works without problems.

What's Included

When you open 'HTML' you will find 7 page templates along with js, css, sass, etc.. folders.

SASS Files

CSS Files

If you don't know how to work with sass or just want to work directly on the stylesheets, we organized the css structure for those who can't work with sass. Any template has a folder named css that contains 5 files.
